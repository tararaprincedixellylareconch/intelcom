Preguntas frecuentes
¿Puedo probar el producto antes de comprarlo?, ¿En qué consiste el crédito dede Google Maps Platform?

Google Maps Platform ofrece un crédito mensual depara Maps, Routes y Places (consulta Créditos de las cuentas de facturación). Algunos clientes descubren que, gracias a este crédito, no deben pagar nada por el uso que hacen de esas herramientas. No se te cobrará hasta que tu uso exceda losen un mes.

Ten en cuenta que, actualmente, la API de Maps Embed, el SDK de Maps para Android y el SDK de Maps para iOS no tienen límites de uso y son sin cargo (el uso de la API o los SDK no se consideran para el crédito demensuales).

Are there any products excluded from the Google Maps Platformmonthly credit?

The monthly Google Maps Platform credit does not apply to Mobility services, Asset Tracking, or Navigation SDK.

¿Puedo usar la prueba sin cargo de USD 300 de Google Cloud Platform con Google Maps Platform?

Sí, puedes hacerlo (consulta Créditos de la cuenta de facturación). En el caso de otros créditos, es posible que se apliquen restricciones (consulta la página Facturación de Google Maps Platform).

¿Por qué se requiere una cuenta de facturación o una tarjeta de crédito? ¿Cómo sé que no me cobrarán?

Te pedimos los datos de tu tarjeta de crédito o cuenta de facturación para cubrir cualquier importe que exceda el crédito mensual de USD 200. Puedes establecer cuotas o límites en tus solicitudes diarias para garantizar que el aumento repentino del uso no supere tu presupuesto ni genere facturas inesperadas. Para obtener información a fin de establecer límites diarios, consulta la página Facturación de Google Maps Platform.

¿Qué es una solicitud?

Las solicitudes pueden ser llamadas a la API o eventos de carga de mapas o panorámicas. Las interacciones del usuario, como el uso del zoom o el desplazamiento lateral, no tienen costo. Consulta la página Facturación de Google Maps Platform.

¿Cuál es el precio después del crédito mensual?

Si deseas obtener información detallada sobre los precios de Maps, Routes y Places, consulta la página Facturación de Google Maps Platform.

¿Soy apto para obtener descuentos por volumen?

Al usar más de 100,000 solicitudes por API al mes, se descuenta automáticamente un 20%. Las empresas que tengan grandes volúmenes de uso superiores a 500,000 solicitudes por API al mes deben comunicarse con nosotros para obtener descuentos por volumen, asistencia al cliente de nivel empresarial o contratos sin conexión.

¿Hay precios especiales para organizaciones sin fines de lucro?

Las startups y las organizaciones sin fines de lucro, de respuesta a emergencias y de medios de comunicación aptas pueden acceder a créditos de Google Maps Platform. Obtén información para solicitarlos

¿De qué manera los clientes pueden comprender su factura y su uso?

Consulta la página Facturación de Google Maps Platform o verifica tu uso actual en Google Cloud Platform Console.

¿Qué monedas están disponibles?

Todos los precios se muestran en USD. El precio se convertirá a tu moneda local durante el proceso de facturación. Consulta una lista de las monedas locales admitidas

¿Qué debo hacer para obtener una clave de API?

Haz clic en el botón Comenzar y completa el flujo para obtener tu clave de API. Esta clave funcionará con cualquiera de las API de los productos que hayas habilitado (Maps, Routes o Places). También puedes obtener una clave de API por medio de Google Cloud Platform Console.

¿Qué otros casos de uso admite Google Maps Platform?

Visita nuestro selector de API a fin de acceder a una lista de los casos de uso más comunes y determinar cuál es la mejor API para tu situación.
